# revlookup
Revlookup é uma ferramenta escrita em Python3 que realiza um DNS Reverso em IPs para a identificação de Subdomínios e novas aplicações

# Instalação e Uso
```sh
git clone https://github.com/yHunterDep/revlookup/
cd revlookup/
chmod +x revlookup
```

# Help do RevLookup
```sh
./revlookup -h
```
